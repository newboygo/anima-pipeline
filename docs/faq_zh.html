常见问题

关于安装设置、ComfyUI连接、LLM配置及故障排除的常见问题解答。 如果你的问题未列出，请在 GitHub Issue 提交反馈。

🔧 安装与启动 2
Q 首次启动前需要准备什么？
安装Python 3.10+，然后使用 pip install -r requirements.txt 安装依赖。 同时准备好可运行的ComfyUI环境及Anima工作流JSON文件。
Q 为什么浏览器打不开应用？
请访问 http://localhost:7860（注意是HTTP不是HTTPS）。检查终端日志，确认启动过程无错误。
⚙️ ComfyUI集成 1
Q 如何解决ComfyUI连接错误？
确认ComfyUI正在运行，并正确设置了地址（通常为 http://127.0.0.1:8188）。 然后在应用设置中运行设置诊断工具。
🤖 LLM集成 2
Q 可以不使用LLM吗？
可以。你可以手动输入提示词标签并生成图片，无需LLM集成。
Q 支持哪些LLM服务商？
支持Gemini API、LM Studio以及兼容OpenAI的API接口。
📱 移动端/远程访问 1
Q 如何用iPhone或iPad访问？
将两台设备连接到同一网络，启动ComfyUI时添加参数 --listen --enable-cors-header，然后在手机浏览器中打开 http://<你的电脑IP>:7860。
🕒 历史与画廊 1
Q 生成历史和缩略图保存在哪里？
生成历史：history/history.db
缩略图：history/thumbs/
